create a ansible-prj project in github

generate and upload an SSH key to the project, no passphrase:

	ssh-keygen -t ed25519 -C "minardi.massimiliano@libero.it"
	
add the public key to github, allow write access.

Upload the project:

	cd ansible-prj
	git init
	git add . && git commit -m "Initial commit"
	git remote add origin git@github.com:maxmin13/ansible-prj.git
	eval `ssh-agent -s`
        ssh-add /home/vagrant/.ssh/id_ansible_prj
        git push origin master
